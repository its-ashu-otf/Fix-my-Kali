# Kali-Linux-Fix

A Little Repository that makes using Kali-linux far more pleasant.

## Installation

Run this script using this

```bash
  git clone https://github.com/its-ashu-otf/SnapD-Fix.git
  
  cd Kali-Linux-Fix

  chmod +x <script_name>.sh

  sudo ./<script_name>.sh
```
