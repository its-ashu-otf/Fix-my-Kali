# SnapD-Fix
A bash Script which fixes snapd for Kali Linux

## Installation

Run this script using this

```bash
  git clone https://github.com/its-ashu-otf/SnapD-Fix.git
  
  cd SnapD-Fix

  chmod +x snapd_fixer.sh

  sudo ./snapd_fixer.sh
```
